# Spec: POS (Ponto de Venda / PDV)

## Problema
Operadores de loja fisica precisam criar pedidos rapidamente no balcão sem depender do WhatsApp.

## Objetivo
Interface de PDV touch-friendly para criar pedidos presenciais com produtos, adicionais, desconto e pagamento.
