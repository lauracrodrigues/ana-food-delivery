# PDV Profissional — Proposal

## Problema
PDV atual é funcional para demo mas não para operação real em restaurante.
Operadores sofrem com: falta de cancelamento de item, desconto sem controle,
promoções não aplicadas, UX lenta demais para rush.

## Objetivo
Transformar o PDV em ferramenta profissional onde:
- Uma venda balcão simples leva < 30 segundos
- Operador não precisa de treinamento longo
- Gerente tem controle sobre descontos
- Promoções e happy hour funcionam automaticamente
- Erros de pedido são corrigíveis sem cancelar tudo

## Métricas de sucesso
- Venda completa (produto → pagamento): < 30 segundos
- Cancelamento de item: < 5 segundos
- Aplicação de promoção: automática, 0 cliques extras
